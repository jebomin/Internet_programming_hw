* 4가지 차이점
1. 스코프(Scope): var 변수는 함수 스코프를 갖고 있음. 즉, 함수 내에서 선언된 var 변수는 함수 내에서만 유효하며, 함수 외부에서는 접근할 수 없습니다. 반면에 let 변수는 블록 스코프를 갖고 있어서, 변수가 선언된 블록({}) 내에서만 유효함. 따라서, let 변수는 해당 블록 내에서만 접근 가능하고 외부에서는 접근할 수 없음.
2. 호이스팅(Hoisting): var 변수는 호이스팅에 영향을 받습니다. 호이스팅은 변수 선언이 해당 스코프의 맨 위로 끌어올려지는 동작을 의미함. 따라서, var 변수는 선언 이전에도 사용할 수 있음. 반면에 let 변수는 호이스팅이 발생하지 않으며, 변수가 실제 선언된 위치 이전에 사용하려고 하면 에러가 발생함.
3. 재선언(Re-declaration): var 변수는 같은 스코프 내에서 중복해서 선언할 수 있음. 이 경우 이전의 값은 덮어쓰여짐. 하지만 let 변수는 같은 스코프 내에서 중복 선언이 허용되지 않음. 중복 선언 시 SyntaxError가 발생함.
4. 전역 변수(Global variable): var로 선언된 변수는 전역 객체(window 객체)의 속성으로 취급됨. 따라서 var 변수는 전역 스코프에서 선언된 경우 어디서든 접근할 수 있음. 반면에 let 변수는 전역 스코프에서 선언해도 전역 객체의 속성이 되지 않고, 전역 스코프 외부에서 접근할 수 없음.
